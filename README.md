# personal-blog

This is a [personal blog](https://roadmap.sh/projects/personal-blog) project using express and json file

To run this,

First, Clone the repo by typing git clone https://github.com/caals666/personal-blog.git

Second, navigate to the folder using cd [Path of folder] in cmdline(for me it is cd C:\Users\Sriram\Documents\code\personal-blog)

Third, Use npm install to install all my depedencies for express

Finally, Use npm start to start the project and open https://localhost:3000
